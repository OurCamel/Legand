A web-based lineup builder for Kuwait National Team fans.

## Features
- Formation selection (4-4-2 / 4-3-3 / 4-2-3-1)
- Interactive player placement
- Position-based validation
- Prevent duplicate player selection
- Save lineup data
- Lineup display page
- X (Twitter) username watermark
- Export lineup as PNG image
- Mobile-first design (iPhone friendly)

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase (Firestore & Hosting)

## How to Use
1. Open the lineup builder page
2. Select a formation
3. Choose players by position
4. Save the lineup
5. View the final lineup
6. Export and share the image

## Notes
This project is optimized for mobile devices and social media sharing.
